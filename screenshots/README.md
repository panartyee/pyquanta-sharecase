# 截图文件夹

将你的文件截图放在这里。

## 建议规格

- **格式**：JPG 或 PNG
- **尺寸**：800x450 像素（16:9 比例）
- **大小**：< 500KB
- **命名**：使用英文和数字，如 `python-tutorial.jpg`

## 没有截图？

如果文件没有截图，可以在 `js/app.js` 中将 `screenshot` 字段留空：

```javascript
screenshot: "", // 留空会显示默认图标
```

## 压缩工具推荐

- [TinyPNG](https://tinypng.com/) - 在线压缩
- [Squoosh](https://squoosh.app/) - Google 出品
- [ImageOptim](https://imageoptim.com/) - Mac 工具

## 免费图床（可选）

如果不想把图片放在网站里，可以使用图床：
- [sm.ms](https://sm.ms/)
- [imgur](https://imgur.com/)
- [路过图床](https://imgtu.com/)

使用图床时，将完整URL填入 `screenshot` 字段即可。
